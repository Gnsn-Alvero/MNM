# MANAVTANI_MAHEK
**💪 Motivation 💪**<br>
There is an NGO for Animal Rescue Named manavta ni mahek which is ran by my friend they had not a website so we made amazing website for them.<br>
**🛠️ Implementation 🛠️**<br>
